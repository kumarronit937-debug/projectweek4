# projectweek4
Tech Solutions Agency

Project Overview

Tech Solutions Agency is a responsive professional services website developed using HTML, CSS, and JavaScript.

The website provides information about the company, its services, and allows users to contact the organization through a contact form.

Features

- Responsive Navigation Bar
- Home Page
- About Page
- Contact Page
- Contact Form Validation
- Mobile-Friendly Design
- Modern User Interface
- Responsive Layout using CSS

---

Setup Instructions

Prerequisites

- Web Browser (Chrome, Edge, Firefox)
- Visual Studio Code (Recommended)

Running the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open "index.html" in your browser.

OR

1. Open the project in VS Code.
2. Install the Live Server extension.
3. Right-click "index.html".
4. Select Open with Live Server.
